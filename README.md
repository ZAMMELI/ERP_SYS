# ERP_SYS – Système ERP Industriel

Projet de système ERP (**Enterprise Resource Planning**) conçu pour gérer les processus métiers dans l'industrie.

## 🎯 Objectif

Développer une application web permettant de gérer :
- La production (ordres de fabrication, nomenclatures)
- La maintenance industrielle
- La gestion de la qualité
- Les ressources humaines
- Le suivi des stocks et achats

## ⚙️ Stack technique

| Couche       | Outil utilisé                         |
|-------------|---------------------------------------|
| Frontend    | Angular + Tailwind CSS                |
| Backend     | Spring Boot + Java + JWT              |
| DB          | Microsoft SQL Server                  |
| Auth        | Spring Security + JWT                 |
| CI/CD       | GitHub Actions + Docker               |
